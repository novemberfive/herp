# 医院资产管理系统 - 接口设计文档（API 规范）

## 1. 文档概述

### 1.1 文档目的
本文档定义医院资产管理系统的 API 接口规范，作为前后端开发的唯一"合同"。每个接口都明确对应到页面上的具体模块和功能状态。

### 1.2 适用范围
- 前端开发工程师
- 后端开发工程师
- 测试工程师
- 接口调用方

### 1.3 术语定义
| 术语 | 定义 |
|------|------|
| API | 应用程序编程接口（Application Programming Interface） |
| REST | 表述性状态转移（Representational State Transfer） |
| JWT | JSON Web Token |
| RBAC | 基于角色的访问控制（Role-Based Access Control） |

### 1.4 接口规范

#### 1.4.1 基础约定
- **协议**: HTTPS
- **基础路径**: `/api/v1`
- **数据格式**: JSON
- **字符编码**: UTF-8
- **时间格式**: `yyyy-MM-dd HH:mm:ss`
- **日期格式**: `yyyy-MM-dd`

#### 1.4.2 认证方式
```
Authorization: Bearer {token}
```

#### 1.4.3 通用响应格式

**成功响应**:
```json
{
    "code": 200,
    "message": "success",
    "data": {},
    "timestamp": 1704067200000
}
```

**失败响应**:
```json
{
    "code": 400,
    "message": "参数错误",
    "data": null,
    "timestamp": 1704067200000
}
```

#### 1.4.4 分页参数
```json
{
    "pageNum": 1,
    "pageSize": 10,
    "sortField": "createTime",
    "sortOrder": "desc"
}
```

#### 1.4.5 分页响应
```json
{
    "code": 200,
    "message": "success",
    "data": {
        "list": [],
        "total": 100,
        "pageNum": 1,
        "pageSize": 10,
        "pages": 10
    },
    "timestamp": 1704067200000
}
```

## 2. 错误码定义

### 2.1 通用错误码

| 错误码 | 说明 | 处理建议 |
|-------|------|---------|
| 200 | 成功 | - |
| 400 | 请求参数错误 | 检查请求参数 |
| 401 | 未授权 | 重新登录 |
| 403 | 禁止访问 | 检查权限 |
| 404 | 资源不存在 | 检查资源 ID |
| 500 | 服务器内部错误 | 联系管理员 |

### 2.2 业务错误码

| 错误码 | 说明 | 处理建议 |
|-------|------|---------|
| 1001 | 用户不存在 | 检查用户名 |
| 1002 | 密码错误 | 重新输入密码 |
| 1003 | 账号已禁用 | 联系管理员启用 |
| 1004 | 登录失败次数过多 | 等待后重试 |
| 2001 | 资产不存在 | 检查资产 ID |
| 2002 | 资产状态不允许此操作 | 查看资产当前状态 |
| 2003 | 资产编号已存在 | 更换资产编号 |
| 2004 | 库存不足 | 先进行采购 |
| 3001 | 采购单不存在 | 检查采购单 ID |
| 3002 | 采购单状态不允许此操作 | 查看采购单状态 |
| 4001 | 盘点任务不存在 | 检查任务 ID |
| 4002 | 盘点任务已结束 | 创建新任务 |
| 5001 | 维修单不存在 | 检查维修单 ID |
| 5002 | 维修单状态不允许此操作 | 查看维修单状态 |
| 6001 | 处置单不存在 | 检查处置单 ID |
| 6002 | 处置单状态不允许此操作 | 查看处置单状态 |

        "avatar": "string, 头像 URL",
        "roles": ["array, 角色列表"],
        "permissions": ["array, 权限列表"]

}
