# 资产领用退库功能开发说明

## 一、功能概述
资产领用退库模块用于管理企业资产的领用申请和退库归还流程，支持完整的审批工作流。

## 二、已完成内容

### 2.1 后端开发
1. **实体类** (`AssetRequisition.java`)
2. **Repository 层** (`AssetRequisitionRepository.java`)
3. **Service 层** (`AssetRequisitionService.java` + `AssetRequisitionServiceImpl.java`)
4. **Controller 层** (`AssetRequisitionController.java`)

### 2.2 前端开发
- 页面文件：`frontend/src/views/management/RequisitionList.vue`
- 功能：搜索、列表、分页、CRUD、审批流程

## 三、数据库表
asset_requisition 表（详见实体类字段）

## 四、API 接口
- GET /management/requisition/list - 分页列表
- GET /management/requisition/{id} - 详情
- POST /management/requisition - 创建
- PUT /management/requisition - 更新
- DELETE /management/requisition/{id} - 删除
- POST /management/requisition/{id}/submit - 提交审批
- POST /management/requisition/{id}/approve - 审批通过
- POST /management/requisition/{id}/reject - 审批拒绝
